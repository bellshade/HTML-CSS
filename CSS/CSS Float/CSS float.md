# CSS FLOAT

Css float adalah property css yang digunakan untuk membuat element mengambang.

### Langsung saja ke kodenya

```css
div {
  /* Element mengambang ke kiri */
  float: left;
}
```

### Contoh float left

![CSS Float](css-float_left.png)

```css
div {
  /* Element mengambang ke kanan */
  float: right;
}
```
